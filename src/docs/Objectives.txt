

The given code is suppose to do this :

1 Program generates 3 random characters at a time

2 Each character is assigned a random class

3 Each character is assigned with class base weapon [ random every time ]

4 Every character has 3 core stats [Power , Health , Speed] which are randomized values with specific range based on class

5 Every character has 2 Sub stats which are corresponding to the class
    - For Melee : ["Armor", "Crit Chance"]
    - Magic class receives an additional mana stat 

6 Character rarity tiers (Common, Rare, Epic, Legendary) affecting char's properties
    - 1 Based on rarity stats will change
    - 2 based on rarity weapons will change
